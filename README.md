# CitiOps Monitor – Application Support Monitoring & Incident Tracker

CitiOps Monitor is a medium-sized project that simulates real-world application support activities.

## Features
- Monitors application and server logs
- Detects ERROR and WARNING events
- Performs basic service health checks
- Generates incident summary reports
- Uses shell scripting for automation

## Technologies
- Python
- Shell Scripting
- Log Monitoring
- Incident Reporting
- Application Support Concepts

## Project Structure
CitiOps-Monitor/
├── logs/
│   ├── app_log.txt

├── scripts/

│   ├── health_check.py
│   └── incident_report.py
├── automation/

├── output/
│   └── support_summary.txt
└── README.md

## Purpose
This project demonstrates monitoring, troubleshooting, and reporting tasks commonly performed in an application support role.
